# D3D11 - RIT Academic Coursework - Fall 2025

Class - Foundations of Game Graphics Programming

I worked in C++, with the DirectX11 API, to build a system that creates a 3D environment. 

We began with a codebase created by our professor, Chris Cascioli, that created a window and put a single triangle up on screen. We quickly progressed through tasks such as:

1) Constructing three dimensional objects from data files
2) Implementing ImGui to debug during runtime, and create tools to manipulate objects
3) Realistically modelling textures and surfaces using Lambert, Phong and PBR lighting models
4) Implementing directional, point, and spot lighting with attenuation
5) Implementing skyboxes, shadows (through shadow maps)
6) Post process effects such as blurring
7) Implementing a camera system

I studied GPUs, ways GPUs map memory during resource creation; understood the, sometimes ridiculous, ways corporate demands drive GPU development; implemented efficient solutions for GPU memory copies such as a ring buffer.

--------------------

This project can run on Windows only. To run, clone the repository locally, and open the solution file in Visual Studio to run. Packages necessary to run should be added automatically.
